# JornaApp (Demo)

Projeto demo em React + Vite com Tailwind CSS. Contém a interface que você forneceu pronta para rodar no navegador.

Como rodar (macOS / zsh):

```bash
# 1) Instalar dependências
cd "/Volumes/Programs - Dados/Projetos/Jornasa"
npm install

# 2) Rodar em desenvolvimento
npm run dev
```

Abra o endereço mostrado pelo Vite no navegador (normalmente http://localhost:5173).

Notas:
- O verificador de fontes é apenas simulado; substitua por sua API quando disponível.
- Para compilar para produção use `npm run build` e `npm run preview`.
